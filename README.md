# 这个项目可以用来使用键盘控制无人机或者通过YOLOv5识别目标点自动操控无人机前往目标点



## 如何使用

### 1. clone这个项目到你本地
```bash
$ git clone https://github.com/Bilibilimoon/FPV_ROS2_Workspace.git
```
### 2. cd进入这个工作空间用colcon命令编译这个项目
```bash
$ colcon build
```
